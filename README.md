# Cybersecurity Awareness Training

A short, interactive eLearning course designed to help employees recognize cyber threats, protect sensitive information, and follow cybersecurity best practices in the workplace.

## 📖 About the Project

Cybersecurity threats continue to evolve, making employee awareness a critical component of organizational security. This training module provides learners with practical knowledge and real-world examples to help prevent security incidents.

The course was developed using **Articulate Storyline 360**.

7. Final Assessment
